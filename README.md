# Quicko Chrome Extension

A Chrome extension to save and manage quick shortcuts for easy access. Quicko allows users to add, search, copy, and delete shortcuts efficiently, improving browsing productivity.

## Features

- **Save Shortcuts**: Add URLs with custom names for quick access.
- **Search Shortcuts**: Easily find saved shortcuts using the search bar.
- **Open Shortcuts**: Click on a shortcut to open the URL in a new tab.
- **Copy URL**: Copy shortcut URLs to the clipboard with a single click.
- **Delete Shortcuts**: Remove shortcuts when no longer needed.
- **Keyboard Shortcut**: Use the default keyboard shortcut **Alt+Q** to open the extension.

## Installation

### 1. Clone or Download the Repository

Clone this repository or download the extension files.

```bash
git clone https://github.com/SD2B/quicko.git
```

### 2. Load the Extension in Chrome

1. Open Chrome and navigate to `chrome://extensions/`.
2. Enable **Developer mode** in the top-right corner.
3. Click **Load unpacked**.
4. Select the folder where you have the extension files.

### 3. Set the Keyboard Shortcut (Default)

The extension uses the default keyboard shortcut **Alt+Q** to open the popup.

## Usage

1. Click the **Quicko** extension icon in the toolbar to open the popup.
2. Use the **Search** field to filter through your saved shortcuts.
3. Click a shortcut to open the corresponding URL in a new tab.
4. Click the **Copy** button to copy a shortcut URL.
5. Click the **Delete** button to remove a shortcut.
6. Use the keyboard shortcut **Alt+Q** to quickly open the extension.

## Technologies Used

- **HTML/CSS**: For creating the popup UI.
- **JavaScript**: For handling shortcut management and interactions.
- **Chrome Extensions API**: For interacting with browser storage.
